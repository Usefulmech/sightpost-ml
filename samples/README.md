# Demo validation images

Place non-private demo images here while calibrating the pitch flow.

Recommended sets:

- `person_a_reference.jpg`: missing-person registration photo.
- `person_a_sighting.jpg`: public sighting photo for the same person.
- `person_a_patient.jpg`: hospital intake-style photo for the same person.
- `person_b_nonmatch.jpg`: different person for false-match checks.
- `no_face.jpg`: bad input case.

Do not commit real missing-person, patient, family, or reporter data.
